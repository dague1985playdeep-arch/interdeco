## 🚀 QUICK START

```bash
# 1. Setup (2 min)
cp .env.example .env
# Editar .env con tus credenciales

# 2. Instalar (1 min)
pip install -r requirements.txt

# 3. Inicializar (1 min)
python init_db.py
python cli.py seed-data

# 4. Ejecutar (< 1 min)
python main.py

# 5. Visitar (instantáneo)
http://localhost:5000
```
